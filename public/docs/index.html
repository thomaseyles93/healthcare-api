Authenticating requests

This API is not authenticated.

Endpoints

Return a patients medical record.

Example request:
curl --request GET \
    --get "http://localhost/api/emis/pfs/patientRecord/iste/qui" \
    --header "Content-Type: application/json" \
    --header "Accept: application/json"
const url = new URL(
    "http://localhost/api/emis/pfs/patientRecord/iste/qui"
);

const headers = {
    "Content-Type": "application/json",
    "Accept": "application/json",
};

fetch(url, {
    method: "GET",
    headers,
}).then(response => response.json());
